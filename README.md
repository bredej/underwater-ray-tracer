# Underwater Ray Tracer

A C++ header-only library that implements a simple ray tracer designed for simulating sound propagation in an underwater environment.

![plot.png](plot.png)

--- 

## Build and run the plot example

To build and run the plot example, you will need [gnuplot](https://www.gnuplot.info/), CMake and a C++ compiler (e.g., g++).

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/bredej/underwater-ray-tracer.git
    cd underwater-ray-tracer
    ```

2.  **Build**  
    On Linux:
    ```sh
    sudo chmod setup.sh +x
    ./setup.sh
    ```
    On Windows:
    ```sh
    ./setup.bat
    ```

3.  **Run**
    ```sh
    ./build/bin/plot
    ```
